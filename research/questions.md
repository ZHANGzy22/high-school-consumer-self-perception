# 开放问题

## 核心研究问题

高中生对自己消费判断能力的自我认知，和他们在模拟消费情境中的实际选择是否一致？

## 可拆分的问题

1. 参与者通常如何描述自己的消费判断能力？
2. 哪些数字消费情境最容易让参与者偏离自己原先的自我判断？
3. 参与者在偏离后如何解释自己的选择？
4. 这些解释更接近预算考虑、真实需要、情绪冲动、害怕错过、信息信任，还是界面忽略？
5. 不同参与者之间是否存在可观察的策略差异，例如信息核查型、预算维持型、即时满足型、事后合理化型？

## MVP 阶段先回答的问题

MVP 阶段先回答一个小问题：当高中生先说自己“会谨慎判断”，再遇到 3 个常见数字消费情境时，他们是否仍然会保持相同的判断方式？

## 暂时不处理的问题

1. 不研究真实消费记录，因为隐私成本高，也超出 Week 3 的可行范围。
2. 不研究长期消费习惯，因为现在的产品还没有多日追踪功能。
3. 不给用户做固定人格诊断，因为标签过早会让项目显得简单。
4. 不把结果解释成“理性/不理性”的二分评价，因为高中生消费选择背后可能有社会环境、预算、情绪和身份因素。

## 需要导师帮助确认

1. 自我认知题目应如何写，才能减少目的性和暗示性。
2. 三个模拟情境应选哪些，才能既贴近日常生活，又有研究价值。
3. 是否需要给不同参与者随机呈现场景顺序。
4. 开放回答如何编码，是否可以使用主题分析。
5. 网页收集用户数据时，是否需要明确说明匿名和用途。
