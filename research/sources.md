# 文献与资料表

研究问题：高中生对自己消费判断能力的自我认知，和他们在模拟消费情境中的实际选择是否一致？

说明：本表用于 Week 3 的 research archive。证据强度分为：强、较强、中、辅助。后续写论文时，优先使用“强”和“较强”的来源。

## 六、目前最适合转化进游戏的机制

1. 限时倒计时：测试用户是否会从“我会慢慢考虑”转向“先下单”。
2. 默认加购：测试用户是否会主动检查附加项。
3. 满减门槛：测试用户是否为了优惠购买原本不需要的东西。
4. 评论冲突：测试用户是否会查更多信息，还是只看高赞/好评。
5. 预算提醒：测试用户是否会在提醒出现后修正选择。

## 七、需要导师确认的问题

1. 我的自我认知题目是否应使用成熟量表的一部分，还是自拟更贴近高中生的中性问题。
2. 模拟情境是否需要随机顺序，还是 MVP 阶段可以固定顺序。
3. 参与者开放回答是否需要匿名处理，以及如何写 consent / data privacy。
4. 结果页面是否能显示“倾向画像”，还是只显示“自我认知与选择的一致/不一致点”。
